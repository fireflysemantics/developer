Generating Google Font Previews with SuperflyCSS | Task

account_circle
Ole Ersoy
Apr - 11  -  10 min

Scenario

We want this fixture rendered with every single Google Font at normal weight.

<h1>The Font Family is {{ font }}</h1>

Approach

Install the SuperflyCSS CLI:

npm i @superflycss/cli

Then create a prototyping project:

sfc new p
cd p

Install the superflycss/utilities-fonts package:

npm i @superflycss/utilities-fonts

Import all Google Font utilities by updating src/test/css/index.css such that it looks like this:

@import "../../main/css/";
@import '@superflycss/utilities-fonts'

Also update src/test/html/index.html such that the body element contains this:
